# Introducción

BuscaRamos es una plataforma creada por y para estudiantes de la Pontificia Universidad Católica de Chile (UC), diseñada para revolucionar la forma en que la comunidad universitaria explora, compara e inscribe cursos.

El objetivo principal es empoderar a los estudiantes con información real, transparente y útil sobre los ramos, permitiendo tomar decisiones informadas y colaborativas.

## Tecnologías y servicios utilizados

BuscaRamos está construido con tecnologías modernas y servicios robustos para asegurar rendimiento, accesibilidad y escalabilidad:

- **Astro**: Framework principal para el desarrollo del frontend y la generación de páginas estáticas e interactivas.
- **React**: Para componentes interactivos y experiencias dinámicas dentro de Astro.
- **Tailwind CSS v4**: Sistema de estilos utility-first, con enfoque mobile-first y breakpoints personalizados (`tablet:`, `desktop:`).
- **D1 (Cloudflare D1)**: Base de datos relacional para almacenar cursos, reviews y estadísticas.
- **R2 (Cloudflare R2)**: Almacenamiento de objetos para comentarios extensos y archivos markdown.
- **NDJSON público**: Para exponer datos agregados de cursos y reviews.
- **AstroActions**: Funciones seguras del lado del servidor para validar, procesar y almacenar datos.
- **Cloudflare Pages**: Para el despliegue y hosting del sitio.

Todo el desarrollo es abierto, comunitario y pensado para ser accesible y fácil de mantener.

## ¿Qué ofrece BuscaRamos?

- **Reseñas auténticas:** Opiniones y experiencias reales de estudiantes sobre los cursos, con un sistema de moderación y almacenamiento seguro.
- **Estadísticas detalladas:** Datos sobre carga académica, asistencia, dificultad y más, presentados de forma clara y visual.
- **Recursos académicos:** Acceso a materiales, guías y recomendaciones compartidas por la comunidad.
- **Navegación intuitiva y accesible:** Interfaz mobile-first, accesible y optimizada para todos los dispositivos.

## ¿Cómo funciona?

1. Busca tu curso usando el buscador inteligente.
2. Explora reseñas, estadísticas y recursos de la comunidad.
3. Comparte tu experiencia y ayuda a otros estudiantes.
